Autel
-----------
Для анонимизации дронов Autel пульт и дрон прошиваются на прошивку [Overlord][ref_overlord] или [Boom][ref_boom].
Прошивки по функционалу похожи но есть важное отличие, Overlord всегда обновляет прошивку дрона до последней официально доступной. Boom работает иначе.
